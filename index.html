About This Dashboard:

This dashboard was built using d3.js, a Javascript data visualization library. All data was scraped off website using tabula and scrubbed with good ol' Excel. The repo can be found here.
Source: State Bank Of Pakistan's quarterly branchless banking newsletters.

About Me
